## Fix Lag Playback
#### Lower Timeline Resolution  
1. Go to Project Settings → Master Settings.
2. Lower the Timeline Resolution (e.g., 4K → 1080p).
3. Save and continue editing.

#### Adjust Playback Proxy Mode  
1. Go to Playback → Proxy Mode.
2. Select Half or Quarter Resolution.
  
#### Enable Render Cache
1. Go to Playback → Render Cache → Smart.
2. Wait for red bars to turn blue.
  
#### Use Optimized Media (Proxies) - notrecomended  
1. Select clips in the Media Pool.
2. Right-click → Generate Optimized Media.
3. Choose ProRes Proxy or DNxHR LB in Preferences.
  
#### Other
- Close other apps
- Upgrade your device
